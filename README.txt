>> What this project is

This project generates beautiful unique cellular automaton. 
best_of_cell.zip is a folder of 21 of my favorite randomly genererated cellular automaton.

>> potential uses of this project.
The pixel size and the dimension can be changed in the code. 
This could be used for custom profile pictures, wallpapers, tapestries, logos. 

>> How to operate this program:
Requires Pyglet and Python 3.
